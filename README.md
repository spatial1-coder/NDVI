# NDVI Mapping using Google Earth Engine

## Project Overview
This project calculates the Normalized Difference Vegetation Index (NDVI) for Andhra Pradesh using Sentinel-2 imagery in Google Earth Engine.

## Tools Used
- Python
- Google Earth Engine
- geemap
- Google Colab
